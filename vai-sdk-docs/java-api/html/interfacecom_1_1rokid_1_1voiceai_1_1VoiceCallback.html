◆ onAccept()

void com.rokid.voiceai.VoiceCallback.onAccept ( String  status)

云端是否接受此次语音激活 仅当前端远场语音激活时有效 如按键录音，不会有此回调。即startVoice(null)的情况下，不会回调

参数
status云端返回的状态 "accept" 激活成功(仲裁) "reject" 拒绝激活(仲裁) "fake" 误激活(云端二次确认)
