语音技能会话

一般来说，用户打开某个技能，直到明确退出或又打开另一个技能，一次会话才算结束。
例如用户"今天天气怎么样"打开了天气技能，音箱播报完今天天气后，会话并不算结束，
用户可以继续问"明天呢"，音箱继续播放天气技能。直到用户说"没事了"等退出指令，
会话才结束。

类成员变量说明

◆ ttsPlayFinish

std::function<void()> rokid::VoiceAI::Session::ttsPlayFinish

tts播放完成，调用此函数通知sdk，sdk将状态上报给云端，
根据具体情况，云端可能会下发下一步的指令。

◆ mediaPlayFinish

std::function<void()> rokid::VoiceAI::Session::mediaPlayFinish

媒体播放完成，调用此函数通知sdk，sdk将状态上报给云端，
根据具体情况，云端可能会下发下一步的指令。

◆ nativeExit

std::function<void()> rokid::VoiceAI::Session::nativeExit

云端技能通过ttsPlayFinish, mediaPlayFinish，sdk能判断出云端技能是否执行完毕。
而本地技能应用执行完毕，需要调用此函数通知sdk，sdk才知道本地技能执行完毕。

◆ isActive

std::function<bool()> rokid::VoiceAI::Session::isActive
