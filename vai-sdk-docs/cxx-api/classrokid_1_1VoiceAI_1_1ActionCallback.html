void *  userp 
)
pure virtual

会话当前所有动作执行完成

这不代表会话结束
例如用户说"今天杭州天气怎么样"，当设备播放完天气tts后，
会话所有动作执行完成，回调onSessionActionCompleted
此时"询问天气"的会话未结束，用户可以再说"明天呢"，设备将继续播放明天杭州天气
当明天天气播放结束，onSessionActionCompleted会再次回调
直到用户说"没事了"/"休息吧"或者用户发起另一个会话
当前会话才结束，回调onSessionEnd
