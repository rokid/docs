1.1. 准备阶段

申请成为智能家居开发者需要准备：

  • 拥有一台若琪设备，可通过 若琪官网(www.rokid.com)或其他线上渠道购买；
  • 如果您是云云对接（推荐）：
    • 在若琪开放平台注册开发者帐号 若琪开放平台；
    • 注册完成后在个人帐号页面行企业认证（如下图），企业认证所需信息：
    • 企业名称
    • 营业执照编号
    • 营业执照照片
  • 如果您是 局域网对接 或 局域网自动发现：
    • 和商务签订芋头科技 IoT 接入协议，详情请咨询对接商务经理， 点击这里 提交表单；

1.2. 厂商接入

不管是云云对接，还是局域网对接，都需要遵循若琪智能家居协议，提供设备搜索接口、设备控制接口、设备状态查询接口（可选）。

我们推荐使用 v2 版本协议完成接入：若琪智能家居协议

1.2.1. 云云对接

对于云云对接，我们遵循 OAuth 2.0 协议接入服务方授权。

您可以参考以下文档完成云云对接：

  • 云云对接
  • OAuth 2.0 协议

1.2.2. 局域网对接

对于局域网对接，我们遵循 SSDP 协议实现局域网内的设备自动发现。

您可以参考以下文档完成局域网对接：

  • 局域网对接
  • 局域网自动发现

1.3. 厂商自测

1.3.1. RHOME 桌面调试工具

使用 RHOME 桌面调试工具，验证设备云服务接口是否正确接入若琪智能家居协议。

您可以参考 RHOME 桌面版调试工具文档 完成调试。

1.3.2. 自定义接入

使用自定义接入可以很方便的调试开发中的远程驱动，目前支持 HTTP 远程驱动，TCP 远程驱动的调试。

您可以参考 自定义接入文档 完成调试。

1.4. 申请上线

自测通过之后，请下载测试用例并按模板填写：测试用例下载

至少经过一周长稳测试，输出测试报告：测试报告下载

将测试用例和测试报告通过邮箱(smarthomeconnect@rokid.com)发送给我们。

1.5. 发布上线

我们会审核表单中信息，并根据测试报告内容进行真机测试验证，预计 1-2 个工作日可完成发布上线。

1.6. 获取帮助

想要获得更多帮助，可以发邮件到如下邮箱(smarthomeconnect@rokid.com)。
